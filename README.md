Add a .env file with
```
DEEPGRAM_API_KEY=

# Optional overrides (defaults shown)
DEEPGRAM_API_URL=https://api.deepgram.com
DEEPGRAM_AUTH_SCHEME=Token
```

To run:
```
uv run python app.py
```

Go to:
http://127.0.0.1:8000/
